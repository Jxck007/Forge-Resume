# Forge Resume Rules

Product:
ATS-first Resume Builder

Priorities:
1. ATS compatibility
2. PDF reliability
3. Mobile responsiveness
4. Accessibility

Stack:
React 19
TypeScript
Vite
Firebase

Constraints:
- Firebase Spark only
- No Cloud Functions
- Vercel deployment

Design:
Linear
Vercel
Raycast

Avoid:
- Glassmorphism
- Purple gradients
- Crypto styling

PDF Requirements:
- Text-based PDFs
- Clickable hyperlinks
- ATS-readable output

ATS Requirements:
- Alias matching
- Evidence-based scoring
- No fake scores
