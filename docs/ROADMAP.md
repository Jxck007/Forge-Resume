# Roadmap

Forge Resume is in beta. This roadmap is intentionally conservative and should not be read as a promise of exact delivery dates.

## Current status

Current production focus:

- resume builder stability
- live preview consistency
- reliable PDF export
- guest/account flow clarity
- safe optional AI assistance
- import beta polish

## Near-term priorities

### Builder and UX polish

- refine editing ergonomics
- improve small-screen behavior
- keep preview/export workflows clear
- continue reducing rough edges in controls and layouts

### Import improvements

- improve extraction quality for difficult PDFs
- strengthen imported content normalization
- improve review clarity before save
- keep import warnings honest and actionable

### Export improvements

- continue PDF reliability work
- tighten long-content rendering behavior
- improve user guidance around export expectations

### Operational hardening

- keep Forge Free Beta AI status messaging clear
- continue quota/abuse protection verification
- improve deployment and troubleshooting guidance

## Medium-term exploration

- stronger parsing diagnostics for imported resumes
- better import cleanup for edge cases
- broader template polish and regression checks
- continued security/privacy hardening for AI-related flows

## ATS status

ATS is **paused / future-facing**.

That means:

- do not treat ATS as a current production feature
- do not market active ATS scoring as live functionality
- future ATS work should be documented clearly as a rebuild or later phase

## Future / optional directions

Potential future work may include:

- richer export options
- stronger parser foundations
- improved renderer flexibility
- a later ATS or job-match rebuild

Only document these as future possibilities until they are active in the product.
